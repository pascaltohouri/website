A Quick Formal Description of Optimal Path Traversals

Nov 21, 2025 • 5 Min Read

The Vector Space on $\mathbb{R}^n$

A vector space is not just an unordered jumble of vectors. The vector space on the reals \(\mathbf{V} = \mathbb{R}^n\) is taken with operations and axioms that ensure the space is well-behaved. For example, if we start at the origin and travel along one vector, then a second vector, pointing in the same direction, we do not expect to return to the origin. So, the sum of two parallel vectors $\vec{u}$,$\vec{v}$ where $\vec{u}$ starts at $\mathbf{0}$, and $\vec{v}$ at $\vec{u}$ should not be $\mathbf{0}$.

These common-sense assumptions are guaranteed by eight axioms that, in the interest of brevity, I will not elaborate on, other than to list them (where $\vec{u}, \vec{v}, \vec{w} \in V$ and scalars $a, b \in \mathbb{R}$):

Vector Addition

  • 01. Associativity: $(\vec{u} + \vec{v}) + \vec{w} = \vec{u} + (\vec{v} + \vec{w})$
  • 02. Commutativity: $\vec{u} + \vec{v} = \vec{v} + \vec{u}$
  • 03. Identity Element: $\exists \mathbf{0}$ such that $\vec{u} + \mathbf{0} = \vec{u}$
  • 04. Inverse Element: $\forall \vec{u}$, there exists $-\vec{u}$ such that $\vec{u} + (-\vec{u}) = \mathbf{0}$

Scalar Multiplication

  • 01. Compatibility: $a(b\vec{u}) = (ab)\vec{u}$
  • 02. Multiplicative Identity: $1 \cdot \vec{u} = \vec{u}$
  • 03. Distributivity (Vector Addition): $a(\vec{u} + \vec{v}) = a\vec{u} + a\vec{v}$
  • 04. Distributivity (Scalar Addition): $(a + b)\vec{u} = a\vec{u} + b\vec{u}$

Paths as Mappings

A path is like a line that resides within the vector space. In the three physical dimensions, a projectile’s trajectory traces a literal path. Think of a plane, rocket, or bird’s flight path. Since we have no direct experience of paths in higher dimensions, we can use the differences between paths in the observed dimensions to conjecture about high dimensional paths.

Starting in one dimension, movement is restricted to a scalar value changing over time. There is no "wiggle room"; one can only move forward or backward along the line. Formalizing this for $n$-dimensions, we define a path $\gamma$ as a correspondence between a path parameter (like time) and the vector space:

$$ \gamma: [a, b] \to \mathbb{R}^n $$ $$ t \mapsto \gamma(t) = (x_1(t), x_2(t), \dots, x_n(t)) $$

Here, $\gamma(a)$ represents the starting point and $\gamma(b)$ the endpoint. This function is continuous, ensuring the path is unbroken, which is a necessary condition for physical traversal.

Optimal Paths

To discuss "optimality," we must first define a criterion for what constitutes the "best" path. Typically, an experimenalist expresses preferences over experiments with a utility or preference function.

$$ U = f(\mathbf{x}_t) $$

In geometry, this is often distance; in physics, it might be action or time. We formalize this cost by assigning a scalar value to every possible path. This mapping from a function space to the reals is called a functional.

The most common cost functional takes the form of an integral over the path's duration. If we denote a cost function $L(\gamma(t), \gamma'(t), t)$—often called the Lagrangian—the total cost $J$ of a path $\gamma$ is:

$$ J(\gamma) = \int_{a}^{b} L(\gamma(t), \gamma'(t), t) \, dt $$

For simple Euclidean length, the cost is merely the magnitude of the velocity vector, $L = \|\gamma'(t)\|$, integrated over time. However, in optimization problems involving terrain or potential fields, $L$ becomes significantly more complex.

Defining Optimality

An optimal path $\gamma^*$ is one that minimizes this functional $J$. Formally, we seek a $\gamma^*$ such that for all valid neighboring paths $\gamma$:

$$ J(\gamma^*) \leq J(\gamma) $$

This minimization problem leads us to the Calculus of Variations. Just as we find local minima in standard calculus by setting the derivative to zero, here we seek a "stationary point" of the functional. This yields the famous Euler-Lagrange equation:

$$ \frac{\partial L}{\partial \gamma} - \frac{d}{dt} \left( \frac{\partial L}{\partial \gamma'} \right) = 0 $$

Solving this differential equation provides the necessary conditions for the path to be optimal.

Notes

  1. The choice of the interval $[a,b]$ is arbitrary but usually normalized to $[0,1]$ for topological proofs. ↩︎
  2. Strictly speaking, we require $\gamma$ to be piecewise differentiable to calculate length. ↩︎
